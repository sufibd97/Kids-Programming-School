
Kids Programming School - Static Website (Colorful kids cartoon theme)
Files included:
- index.html
- about.html
- courses.html
- admissions.html
- registration.html
- teachers.html
- faq.html
- gallery.html
- contact.html
- css/style.css
- js/script.js
- assets/logo.svg

How to use:
1. Unzip the files.
2. Open index.html in a browser to view locally.
3. To host on GitHub Pages: create a repo, push files to the repo, enable Pages from main branch (root).
4. To host on Netlify: drag & drop the unzipped folder to Netlify Sites dashboard or connect GitHub repo.
